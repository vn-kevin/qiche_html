中昊晨光化工研究院有限公司（以下简称晨光院）地处川南腹地，沱江之滨，是我国有机氟、有机硅等高分子新材料的重要研发和生产基地，是“自贡国家新材料产业化基地”的核心企业。
自1965年建院至今，经过几代晨光人的艰苦奋斗，先后取得科研成果500余项，专利授权90余项，为我国卫星、火箭、潜艇、“神舟”、“嫦娥”等尖端科学工程作出了重大贡献。
1999年转制以来，按照“寓军于民、军民结合”的指导思想，晨光院大力推进自有技术产业化进程，走出了一条“出技术、出成果、出人才、出效益”的中兴之路， 产值和销售收入突破“双二十亿”，为后续发展奠定了坚实基础。
多年来，晨光院秉承“敢为人先、只争朝夕”的创新精神和“科技兴院、产业报国”的治院方针， 经历10多年的产业化发展，40多年的技术沉淀不断得以释放，使企业快速步入绿色、稳健、和谐的可持续发展之路，先后被认定为“国家认定技术中心”、“国家高新技术企业”、 “国家创新型企业”、“中国自主创新能力行业十强企业”、“全国知识产权示范单位”。“晨光CHENGUANG及图”被认定为“中国驰名商标”。
